# Cinema Website Project

Этот проект представляет собой веб-сайт для кинотеатра, разработанный с использованием HTML, CSS и JavaScript.

## Структура проекта

- **index.html**: Главная страница с популярными и рекомендуемыми фильмами.
- **pages/catalog.html**: Каталог всех доступных фильмов.
- **pages/about.html**: Информация о компании.
- **pages/contacts.html**: Страница с контактной информацией и формой обратной связи.
- **css/**: Стили (reset, base, layout, components).
- **js/script.js**: Логика приложения (рендеринг карточек, валидация форм).

## Функционал

### HTML/CSS
- Семантическая верстка.
- Адаптивный дизайн (Flexbox).
- Единая стилизация через разделенные CSS файлы.

### JavaScript
- **Рендеринг карточек**: Фильмы загружаются из массива объектов и отображаются на странице.
- **Фильтрация**: На главной странице фильмы разделены на "Популярные" и "Рекомендации".
- **Валидация формы**: На странице контактов реализована проверка полей (имя, email, сообщение, согласие) с выводом ошибок.

## Запуск

Просто откройте `index.html` в браузере.
